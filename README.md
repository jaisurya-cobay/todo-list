# Todo List

A simple, offline-friendly todo list app. No build step—just open `index.html` in a browser.

## Features

- Add, complete, and delete tasks
- Double-click to edit (Enter to save, Esc to cancel)
- Filter: All / Active / Completed
- Toggle all · Clear completed
- Data saved in the browser (localStorage)

## Run

- Open `index.html` in your browser, or
- From this folder: `python3 -m http.server 5173` then visit http://localhost:5173

## Files

- `index.html` — markup and structure
- `styles.css` — layout and theme
- `app.js` — logic and localStorage persistence
